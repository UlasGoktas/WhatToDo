What To Do is a simple To-do app written in Swift with Core Data.

* [Swift 5](https://swift.org/blog/swift-5-released/)

<!-- GETTING STARTED -->
## Getting Started

### Pods
* [SwiftLint](https://cocoapods.org/pods/SwiftLint)

### Prerequisites

* iOS 13.0+
* Xcode 11.0+
* Swift 5.0+
* Cocoapods

  ```sh
  sudo gem install cocoapods
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/UlasGoktas/ToDoApp.git
   ```
2. Install Pods
   ```sh
   pod install
   ```
3. Open Xcode workspace
   ```sh
   open ToDoApp.xcworkspace
   ```
